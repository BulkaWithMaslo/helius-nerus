Инструкция по использованию.
Переводы хранятся в JSON-файлах.
Переводы ДОЛЖНЫ храниться внутри папки StreamingAssets внутри папки Assets.
Перевод должен называться в формате:
*Language_name*.json
Список названий языков можно найти тут: https://docs.unity3d.com/ScriptReference/SystemLanguage.html
Программа предполагает обязательное наличие перевода English.json.
Переводы хранятся в формате ключ-значение.
Для разных языков ключи обязаны быть одинаковыми и быть расположены в одинаковом порядке.
Для использования есть 2 варианта:
1) К любому компоненту TextMeshProUGUI (Text Mesh Pro text), текст которого не меняется в ходе работы программы или всегда инициализируется одинаковым значением, можно подвесить компонент LocalizedText. В его единственное поле надо вписать ключ (не значение, а ключ) перевода.
2) В скрипте обратиться к LocalizationManager.Instance.GetLocalizedValue(string key) и забрать свое значение.
Скрипт LocalizationManager надо повесить на какой-то объект, который висит на самой первой игровой сцене.
Инструмент для работы с переводами - специальное окно (вкладка Windwow - Localized Text Editor).
В нем можно открыть или создать файл, сохранить его и писать в сериализованные поля.
Пример:
Ключ - "start_game"
Английский - "Start game"
Русский - "Начать игру"
